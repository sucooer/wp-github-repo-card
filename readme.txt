在 WordPress 中优雅地展示 GitHub 仓库卡片。

== Description ==

WP GitHub Repo Card 允许你在 WordPress 文章和页面中轻松嵌入 GitHub 仓库卡片。

== Installation ==

1. 上传插件文件到 `/wp-content/plugins/` 目录
2. 在 WordPress 后台激活插件
3. 在区块编辑器中使用 "GitHub 仓库卡片" 区块
